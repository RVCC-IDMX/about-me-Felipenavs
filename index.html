Image Hover Effects

Having an interactive website is crucial, most modern websites nowadays have some sort of interactivity because it helps to keep our users more engaged with our content and overall it provides a more enjoyable experice. Image hover effects is a fantastic tool to help us achieve that interactivity. There are many different elements and attributes that we can be use to style and embelish a website using only CSS. Here I'll talk about some of the image hover effects I used on my website.

First, I started by creating a div for each one of my images inside of my gallery section. Inside each of those divs, I placed an image and another div right below it to add the text content I wanted to describe my images. This allowed me to use position relative and absolute to place my text content on top of my image:

Secondly, I styled my content div. I added some backgorund color and some transparency so I could still see the image under it. Also, centered the text and adjusted the font size to my liking:

Now it was time to add some inteactivity to the images. I started by selecting my image and the content div at the same time in css and added a transition property with a delay of 2.5s to them making it so that all the properties later added can transition at the same time:

The first style of animation that I added to my images was opacity. In order to do that I selected my div content and set the opacity to 0 as default. Then I added the pseudo-class :hover to my div content. By doing so, when the mouse hovers over the image the content div shows up and when the mouse isn't hovering it the text content fades out:

After that, I kind of wanted the hovered image to stand out a bit more, so I added more animation to them. I used the 'transform'property with the 'scale' property to make to make the hovered image stand out even more when the mouse is over it. So when the mouse is over the image, the image zoom in a bit:

Lastly, I didn't want the text content to just abruptly appear out of nowhere, I kind of wanted it to slide in from on of the side of the image. To achieve this I used the 'transform' attribute with the 'trasnlate' property. I did 3 differents slide in affects, one for each of my images.

The first one slides in from left to right. So I used the 'trasnlateX' property, and set it to be -100% by default, and when the mouse hover over it to translate to 0%. This way, the text content will appear from the left and take the whole space of the image:

With the second image I used the same transalte property, but from the Y axis -translateY-. My intention was for it the slide from the top to bottom. So I set the first value to 100% and then translateY to 0%:

For my last image i used the translateX property again but I made it slide in from the right side. so I set the first value to 100% and the second to 0%:
